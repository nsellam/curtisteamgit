§ EXTI_QuickInit()

int EXTI_QuickInit ( GPIO_TypeDef *  GPIOx,
uint16_t  pin,
EXTITrigger_TypeDef  trigger,
uint8_t  priority 
)

Makes the initialization of the given external interrupt (EXTI) with the parameters specified.

Parameters
GPIOxPort of the EXTI
pinNumber of the pin of the EXTI
triggerWay to trigg this EXTI
priorityPriority of the exti to initialize
Return values
int(error detected while computing initialization)
Returns
If everything went right EXTI_NO_ERROR, if not EXTI_ERROR_INVALID_LINE, EXTI_ERROR_INVALID_CHANNEL, EXTI_ERROR_INVALID_PORT_SOURCE or EXTI_ERROR_INVALID_PIN_SOURCE.
