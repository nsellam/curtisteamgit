§ ADC_Example()

void ADC_Example ( void  )

Tests all the possible gpio pins as analog inputs for ADC1 Start the debugger and run the program. To test the ADC pins: 1) Connect every ADC pin (PA0..7, PB0..1, PC0..5) to the ground 2) Connect every ADC pin (PA0..7, PB0..1, PC0..5) to the 3.3V Display meaning:

  • 'OK' means the pin passed the test
  • '-' means the pin did not pass the test (yet) Two 'OK' means the pin is very likely to work. Although this test is not 100% reliable, it is a good tool to detect a malfunctioning pin. /!\ PA2 & PA3 are both connected to the USART for the ST_LINK debugger, these pins should not work.
    Return values
    None
